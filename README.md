# DoppelGround Foundation

**The NEXUS OS company and infrastructure landing page.**

A sovereign multi-agent operating system orchestrating AI agents across Slack, Telegram, GitHub, and 280+ tools with vector memory, Kafka event bus, and autonomous decision-making.

## Tech Stack

- **Built with:** Vite + React + TypeScript + Tailwind CSS
- **Hosted on:** Zo Computer (Zo Site)
- **Live at:** https://doppelground-specimba.zocomputer.io
- **Dashboard branch:** https://github.com/specimba/nexusalpha/tree/DASHBOARD-GLM51

## Agent Mesh

| Agent | Role |
|-------|------|
| Zo Computer | Core AI infrastructure |
| AutoClaw | Telegram bot swarm |
| KiloClaw-REBORN | Research & intelligence |
| Grok | Real-time web intelligence |
| Composio | 250+ tool integrations |
| Hyperbrowser | Browser automation |
